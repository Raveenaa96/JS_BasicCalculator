JS Strings

1. at() - The at() method returns an indexed character from a string. Can also use negative index

Example: str.at(-5)

2. charAt() - charAt() returns the character at a specified index (position) in a string

Example: str.charAt(3)

3. toUppercase() - The toUpperCase() method converts a string to uppercase letters

Example: str.toUppercase()

4. toLowerCase() - The toLowerCase() method converts a string to lowercase letters.

Example: str.toLowerCase()

5. length() - length property returns the length of a string

Example: str.length

6. slice() - slice() extracts a part of a string and returns the extracted part.It accept negative indices/indexes

Example: str.slice(1,5)

7. substring() - The substring() method extracts characters, between two indices (positions), from a string, and returns the substring. Treats any negative index as 0

Example: str.substring(1,4)

8. substr() - The substr() method extracts a part of a string.It begins at a specified position, and returns a specified number of characters

Example: str.substr(2,5)

9. repeat() - The repeat() method returns a string with a number of copies of a string

Example: str.repeat(3)

10. replace() - replace() searches a string for a value,and returns a new string with the specified value(s) replaced

Example: str.replace("hello","Javascript")

11. trim() - The trim() method removes whitespace from both sides of a string

Example: str.trim(), can also use trimStart() and trimEnd()

12. split() - split() splits a string into an array of substrings, and returns the array

Example: str.split(" ",3) ,can also use split("") to split characters including spaces

13. indexOf() - The indexOf() method returns the position of the first occurrence of a value in a string. Returns -1 if the value is not found

Example: str.indexOf("e",5) which finds occurence of "e" starting at position of 5, can use indexOf("e") which returns index of first occurrence

14. includes() - The includes() method returns true if a string contains a specified string

Example: str.includes("Hello")

15. lastIndexOf() - lastIndexOf() returns the index of the last occurrence of a specified value in a string

Example: str.lastIndexOf("World")
